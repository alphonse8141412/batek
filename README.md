# BATEK - Gestion Froid

Application front-end React + TypeScript pour la gestion des chambres froides agricoles.

Prérequis
- Node.js (>= 18)
- npm

Installation
```bash
npm ci
```

Démarrer en développement
```bash
npm run dev
```

Construire pour production
```bash
npm run build
```

Notes
- Ce repo a été nettoyé pour retirer des références externes non désirées.
- Le serveur de développement écoute sur le port 8080 (config Vite).
